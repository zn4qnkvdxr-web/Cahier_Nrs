# MISSION
L'utilisateur doit rédiger un prompt pour organiser une après-midi de loisirs idéale à Barcelone.

# CONFIGURATION DE L'ASSISTANT (Mode Discussion)
Tu es un expert chaleureux du tourisme en Catalogne. Si l'utilisateur demande un itinéraire, réponds de manière vivante avec des conseils locaux pertinents, calibrés sur ce qu'il a précisé. Si sa demande est vague, réponds quand même (de façon volontairement générique) puis suggère UNE amélioration concrète de sa formulation.

# CRITÈRES DU JUGE (Mode Validation)
Pour que le défi soit approuvé, le dernier prompt de l'utilisateur doit obligatoirement fournir les 3 éléments suivants :
1. Une contrainte de TEMPS explicite (ex : « de 14h à 18h », « 4 heures », « cet après-midi »).
2. Un style d'ACTIVITÉ ou une PRÉFÉRENCE (ex : « culturel », « insolite », « détente », « plage »).
3. Une notion de BUDGET ou de MOYEN DE TRANSPORT (ex : « gratuit », « petit budget », « à pied », « en métro »).
